#  DESKRIPSI PROJEK
Proyek ini adalah game Catur (Chess) digital yang dirancang sebagai permainan papan klasik dua pemain dengan antarmuka modern dan sederhana. Game ini memungkinkan pemain memainkan catur sesuai aturan standar internasional, dengan fokus pada kemudahan penggunaan dan pemahaman permainan.
- Gambaran Umum
Game menampilkan papan catur 8Ã—8 lengkap dengan bidak Putih dan Hitam. Sistem secara otomatis mengatur giliran pemain, sehingga pemain mengetahui kapan harus melakukan langkah. Antarmuka dibuat minimalis dengan warna papan yang kontras agar bidak mudah dikenali.
- Fitur Utama
Papan Catur Interaktif
Pemain dapat memilih dan memindahkan bidak sesuai aturan catur.
- Indikator Giliran Pemain
Menampilkan informasi giliran (contoh: Giliran: Putih), sehingga tidak terjadi kesalahan langkah.
- Riwayat Gerakan
Menyimpan daftar langkah yang telah dilakukan oleh pemain untuk keperluan evaluasi atau pembelajaran.
- Tombol Undo
Memungkinkan pemain membatalkan langkah terakhir jika terjadi kesalahan.
Game Baru
Mengatur ulang papan dan memulai permainan dari awal.

# HOW TO USE
Permainan catur dimulai secara otomatis saat game dibuka, dengan pemain Putih mendapatkan giliran pertama. Pemain dapat memilih bidak sesuai giliran dengan mengkliknya, lalu kotak yang menyala akan menunjukkan langkah yang dapat dilakukan. Bidak dapat dipindahkan dengan mengklik kotak tujuan atau dengan menyeret (drag) bidak ke posisi yang diinginkan. Informasi giliran dan status permainan seperti Check, Checkmate, atau Stalemate ditampilkan di bagian atas layar, sementara setiap langkah yang dilakukan akan tercatat pada Riwayat Gerakan di sisi kanan. Pemain dapat membatalkan langkah terakhir menggunakan tombol Undo atau memulai ulang permainan dengan tombol Game Baru. Permainan berakhir ketika terjadi Checkmate atau Stalemate.
